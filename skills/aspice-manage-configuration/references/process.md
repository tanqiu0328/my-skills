# SUP.8 配置管理

依据：[Automotive SPICE PAM 4.0 中文版](../../../references/AutomotiveSPICE_PAM_40_Chinese.pdf) 第 90～92 页

## 目的

建立和维护配置项及基线的完整性，并使其对受影响方可用

## 必须覆盖的成果

- 定义选择准则并识别配置项
- 定义配置项的状态、位置、权限等属性
- 建立配置管理及并行修改控制机制
- 控制配置项修改
- 建立内部和交付基线
- 记录、汇总和沟通配置状态
- 确保配置项及基线完整一致
- 验证备份和恢复机制可用

## 证据重点

- 配置项选择准则和清单
- 分支、合并、评审、访问和状态控制
- 基线范围、版本和构建输入
- 修改历史和配置状态
- 完整性检查及备份恢复验证
