# SWE.5 软件组件验证与集成验证

来源：Automotive SPICE PAM 4.0 中文版，第 58～61 页

## 目的

验证组件与架构一致，集成软件要素，并验证集成结果与架构及详细设计一致

## 必须覆盖的成果

- 按架构、详细设计、接口和交互定义集成验证措施
- 为软件组件行为和接口定义组件验证措施
- 按顺序和前提条件集成软件要素
- 按发布范围和回归准则选择验证措施
- 执行组件验证和集成验证并记录结果
- 保持设计、验证措施和结果之间的一致性及双向追溯
- 汇总结果并与受影响方沟通

## 验证重点

- 接口数据的正确解释与传递
- 组件动态交互和时序依赖
- 软件组件行为
- 资源消耗目标
- 每个集成步骤的准入、准出和环境
