# SWE.4 软件单元验证

来源：Automotive SPICE PAM 4.0 中文版，第 56～58 页

## 目的

验证软件单元与软件详细设计一致

## 必须覆盖的成果

- 为详细设计中的每个软件单元定义验证措施
- 按发布范围和回归准则选择验证措施
- 执行静态分析、代码评审、单元测试或适用组合并记录结果
- 保持单元、验证措施和结果之间的一致性及双向追溯
- 汇总结果并与受影响方沟通

## 每项验证措施

- 验证技术与对象
- 通过和失败准则
- 准入和准出准则
- 所需基础设施与数据
- 选择依据与回归准则
- 执行版本、结果和原始证据

偏离预期的结果连接 SUP.9 问题记录
